# 자바 온라인에서 실습하기

 **JDoodle**(https://www.jdoodle.com/)을 이용

\1.  JDoodle 열기 및 Java 언어 선택

\2. 편집기 **main** 구역 내부에 코드를 작성하여 원하는 기능을 구현 

아규먼트를 **CommandLine Arguments** 부분에 입력할 수

\3. **Execute** 버튼을 클릭하여 코드를 실행
